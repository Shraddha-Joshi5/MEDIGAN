# MEDIGAN

A lightweight Streamlit application that uses Medigan to generate synthetic medical images for research, training, and data augmentation.
This app provides an easy web-based interface for selecting models, configuring parameters, and generating medical images instantly.

## Features

- Generate synthetic images using Medigan models

- Supports multiple modalities (MRI, CT, X-Ray, Ultrasound, etc.)

- Simple UI built with Streamlit

- Adjustable generation parameters

- Real-time preview of generated images

- Downloadable image output

Run "streamlit run app.py --browser.serverAddress 0.0.0.0 --server.port 9000" in terminal
